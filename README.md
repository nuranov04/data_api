# data_api
Для запуска проекта вам необходимо сделать следущие команды:


-git clone https://github.com/nuranov04/data_api


-cd data_api


-python3 -m venv env


-source env/bin/activate


-pip install -r requerements.txt


-./manage.py migrate


-./manage.py runserver
